# Sentiment Analysis of Customer Reviews

This repository contains an AI mini-project that applies Natural Language Processing (NLP)
techniques to classify restaurant customer reviews as positive or negative.

## Models Implemented
- Naive Bayes classifier (implemented from scratch)
- Logistic Regression classifier (using scikit-learn)

## Dataset
Restaurant Reviews dataset from Kaggle:
https://www.kaggle.com/datasets/d4rklucif3r/restaurant-reviews

## Features
- Text preprocessing (lowercasing, punctuation removal, stopword removal)
- Bag-of-Words feature representation
- Model evaluation using accuracy, precision, recall, and F1-score

## Tools
- Python
- scikit-learn
